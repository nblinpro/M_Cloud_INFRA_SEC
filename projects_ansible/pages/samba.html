Fiche Mémo : Partage Samba (Ubuntu 24.04)

Partie 1 : Création partage pour un utilisateur

Étape 1 : Installer Samba

Installation des paquets nécessaires :

sudo apt update
sudo apt install samba -y

Étape 2 : Créer le dossier à partager

Création du dossier, attribution des droits et autorisation de traversée du dossier personnel :

# Création du dossier
mkdir -p ~/documents_partages

# Attribution des droits à l'utilisateur courant
sudo chown $USER:$USER ~/documents_partages

# Autorisation d'accès au dossier personnel (traversée)
sudo chmod 755 /home/votre_nom_utilisateur

Étape 3 : Configurer Samba

Ajouter ce bloc à la fin du fichier /etc/samba/smb.conf :

[Documents]
comment = Partage Documents Ubuntu
path = /home/votre_nom_utilisateur/documents_partages
read only = no
browsable = yes
valid users = votre_nom_utilisateur
create mask = 0755
directory mask = 0755

Étape 4 : Définir un mot de passe Samba

Ce mot de passe est indépendant du mot de passe système :

sudo smbpasswd -a $USER

Étape 5 : Redémarrer le service

sudo systemctl restart smbd

Partie 2 : Ajout de plusieurs utilisateurs au partage

Étape 1 : Ajout d’un autre utilisateur

Création de l'utilisateur système et de son mot de passe Samba :

# Création de l'utilisateur système
sudo adduser deuxieme_utilisateur

# Création de son mot de passe Samba
sudo smbpasswd -a deuxieme_utilisateur

Étape 2 : Mettre à jour la configuration

Dans /etc/samba/smb.conf, modifiez la ligne valid users :

[Documents]
comment = Partage Documents Ubuntu
path = /home/votre_nom_utilisateur/documents_partages
read only = no
browsable = yes
# On ajoute le deuxième utilisateur à la suite avec un espace
valid users = votre_nom_utilisateur deuxieme_utilisateur
create mask = 0755
directory mask = 0755

Étape 3 : Redémarrer le service

sudo systemctl restart smbd

Partie 3 : Création d’un second partage

Étape 1 : Création du dossier

mkdir -p ~/nom_repertoire
sudo chown $USER:$USER ~/nom_repertoire

Étape 2 : Configurer Samba

Ajoutez un nouveau bloc. Attention : le nom entre crochets doit être unique.

[Nouveau_Partage]
comment = Autre Partage
path = /home/votre_nom_utilisateur/nom_repertoire
read only = no
browsable = yes
valid users = votre_nom_utilisateur
create mask = 0755
directory mask = 0755

Étape 3 : Redémarrer le service

sudo systemctl restart smbd
