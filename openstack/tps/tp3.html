1. Gestion des Containers

Un container Swift est l'unité de regroupement des objets, similaire à un bucket S3.

CLI - Opérations sur les containers
# Créer un container
openstack container create mon-container

# Lister tous les containers
openstack container list

# Afficher les détails d'un container
openstack container show mon-container

# Supprimer un container (vide)
openstack container delete mon-container

2. Manipulation des Objets

Les objets sont les fichiers stockés à l'intérieur des containers, accessibles via une clé unique.

CLI - Upload & Download
# Uploader un fichier
openstack object create mon-container fichier.txt

# Lister les objets d'un container
openstack object list mon-container

# Afficher les métadonnées d'un objet
openstack object show mon-container fichier.txt

# Télécharger un objet
openstack object save mon-container fichier.txt

# Supprimer un objet
openstack object delete mon-container fichier.txt

3. Contrôle d'Accès (ACLs)

Swift permet de contrôler précisément qui peut lire ou écrire dans un container.

CLI - Gestion des ACLs
# Rendre un container public (lecture)
openstack container set --property "X-Container-Read=.r:*" mon-container

# Autoriser un utilisateur spécifique en lecture
openstack container set --property "X-Container-Read=projet:utilisateur" mon-container

# Autoriser un utilisateur en écriture
openstack container set --property "X-Container-Write=projet:utilisateur" mon-container

# Supprimer les ACLs publiques
openstack container unset --property "X-Container-Read" mon-container

4. Configuration Avancée — TempURL

Les TempURLs permettent de partager un objet de manière sécurisée et temporaire, sans exposer les credentials.

CLI - Génération de TempURL
# Définir une clé TempURL sur le compte
openstack object store account set --property "Temp-URL-Key=ma-cle-secrete"

# Récupérer l'endpoint Swift
openstack catalog show object-store

# Générer une URL temporaire (valide 3600 secondes)
swift tempurl GET 3600 /v1/AUTH_<project-id>/mon-container/fichier.txt "ma-cle-secrete"

5. Gestion des Quotas

Administration des limites de stockage par projet pour contrôler la consommation des ressources.

CLI - Quotas
# Afficher les statistiques du compte
openstack object store account show

# Définir un quota en octets (ex: 1 Go)
openstack object store account set --property "X-Account-Meta-Quota-Bytes=1073741824"

# Définir un quota en nombre d'objets
openstack object store account set --property "X-Container-Meta-Quota-Count=1000"
